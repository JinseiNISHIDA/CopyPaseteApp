# ANTIGRAVITY.md (Project Constitution)

## プロジェクト概要
Windows向けクリップボード履歴管理アプリ（Macアプリ「Paste」風）の開発

## 1. 目的・ターゲット・利用シーン
- **目的:** Windows標準のクリップボード履歴機能の、より洗練された実用的な代替アプリの作成
- **ターゲット:** Windowsユーザー全般
- **利用シーン:** 日常のあらゆるPC作業中（常駐して使用）

## 2. MVP（最小構成）の範囲
- クリップボード履歴の取得（テキスト・画像両対応）
- 履歴の視覚的表示（画面下部へのポップアップ表示など、Paste風のUI）
- 選択項目からのペースト機能
- お気に入り保存機能（よく使うアイテムのピン留め・グループ化）
※「検索機能」「Paste Stack」は不要

## 3. 技術スタック
- コア: HTML / Vanilla CSS / Vanilla JS
- アプリケーションフレームワーク: Electron
  - Windows環境での安定したシステムフックとリッチなUI（アニメーション等）を実現するため。

## 4. 現在のステップ
- **Step 1:** 要件ヒアリングと要件定義（完了）
- **Step 2:** 実装計画（MVP）の策定と承認（y/n） ← 現在ここ
- **Step 3:** MVPの実装（Electron基盤構築、UI、クリップボードロジック）
- **Step 4:** 動作確認（味見）
- **Step 5:** UI/UXのブラッシュアップ

## 5. 設計ルール（備忘録）
1. 命名規則: lower_snake_case.js, lowerCamelCase(関数・変数), UPPER_SNAKE_CASE(定数)
2. DOM参照は `el`、ハンドラは `on` プレフィックスを使用
3. 定数は `config.js` に集約し、コメントで用途を明記
4. UI層とロジック層の分割、適切なエラーハンドリング
